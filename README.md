# Простой текстовый ротокол
`SET;key;value` - сохранить значение  
`GET;key` - взять значение  
`DEL;key` - удалить значение  

# Структура
**server-app/** - Пример использования сервера
**client-app/** - Пример использования клиента
**src/** - Реализация

# Как собрать?
mkdir build && cd build  
cmake ..  
make  

# Как запустить?

-Сервер:  
./server-app/server_app

-Клиент:  
./client-app/client_app
